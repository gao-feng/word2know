# 硅基流动翻译API集成说明

## 功能特点

本插件已集成硅基流动翻译API，提供更准确、更详细的翻译服务：

- 🎯 **高质量翻译**：基于先进的AI模型，提供更准确的翻译结果
- 📚 **详细释义**：包含词性、多种释义、例句等详细信息
- 🔄 **智能回退**：如果硅基流动API不可用，自动回退到Google翻译
- 💾 **本地缓存**：翻译结果本地缓存，提高响应速度
- 🌐 **多语言支持**：支持中文、英文、日文、韩文等多种语言

## 设置步骤

### 1. 获取硅基流动API密钥

1. 访问 [硅基流动官网](https://siliconflow.cn)
2. 注册并登录账户
3. 进入控制台，创建API密钥
4. 复制生成的API密钥

### 2. 配置插件

1. 点击浏览器工具栏中的插件图标
2. 在弹出窗口中选择"设置"标签
3. 确保"翻译服务"选择为"硅基流动"
4. 在"硅基流动API设置"区域：
   - 将API密钥粘贴到输入框中
   - 点击"测试连接"验证密钥是否有效
   - 点击"保存"保存设置

### 3. 开始使用

1. 在任意网页上选中英文单词或短语
2. 点击出现的🌐翻译按钮
3. 查看详细的翻译结果，包括：
   - 中文翻译
   - 音标发音
   - 详细释义（词性、含义、例句）
   - 同义词
   - 常用短语

## 翻译结果示例

使用硅基流动API翻译"beautiful"的结果：

```
beautiful /ˈbjuːtɪfəl/
中文：美丽的，漂亮的

详细释义：
1. adj. 美丽的，漂亮的
   例：She has beautiful eyes.
2. adj. 出色的，完美的
   例：That was a beautiful performance.

同义词：lovely, gorgeous, attractive
常用短语：beautiful day, beautiful music, beautiful view

来源：SiliconFlow
```

## 故障排除

### API密钥无效
- 检查密钥是否正确复制
- 确认账户是否有足够的额度
- 检查网络连接是否正常

### 翻译失败
- 插件会自动回退到Google翻译
- 检查是否被网络防火墙阻止
- 尝试重新保存API密钥

### 连接超时
- 检查网络连接
- 稍后重试
- 可以临时切换到Google翻译服务

## 技术特性

- **模型**：使用Qwen/Qwen2.5-7B-Instruct模型
- **缓存**：本地缓存翻译结果，避免重复请求
- **超时**：4秒超时保护，确保用户体验
- **错误处理**：完善的错误处理和回退机制

## 隐私保护

- 翻译请求直接发送到硅基流动API
- 不会收集或存储用户的翻译内容
- API密钥仅存储在本地浏览器中
- 支持清除本地缓存
