# Anki 导入指南

本插件支持将生词表导出为Anki可导入的CSV格式，方便用户在Anki中进行记忆训练。

## 导出步骤

1. **收集生词**：使用插件翻译单词时，点击⭐按钮将单词添加到生词表
2. **导出文件**：在插件弹窗的"生词表"标签页中，点击"📤 Anki"按钮
3. **下载CSV**：系统会自动下载一个以日期命名的CSV文件（如：vocabulary_2024-01-15.csv）

## Anki 导入步骤

### 1. 打开Anki并创建牌组
- 启动Anki应用
- 点击"创建牌组"，命名为"英语生词"或其他你喜欢的名称

### 2. 导入CSV文件
- 点击"文件" → "导入"
- 选择刚才下载的CSV文件
- 在导入对话框中进行以下设置：

### 3. 导入设置
- **字段分隔符**：选择"逗号"
- **文本编码**：选择"UTF-8"
- **字段映射**：
  - 字段1 → 正面（英文单词）
  - 字段2 → 背面（中文翻译 + 发音）
  - 字段3 → 标签（可选）

### 4. 完成导入
- 点击"导入"按钮
- Anki会显示导入成功的卡片数量

## CSV 文件格式说明

导出的CSV文件包含三列：
```csv
"单词","翻译<br><br><i>发音</i>","vocabulary english"
"beautiful","美丽的；漂亮的<br><br><i>/beautiful/</i>","vocabulary english"
"wonderful","精彩的；绝妙的<br><br><i>/wonderful/</i>","vocabulary english"
```

- **第一列**：英文单词（显示在卡片正面）
- **第二列**：中文翻译和发音（显示在卡片背面，支持HTML格式）
- **第三列**：标签（用于分类管理）

## 学习建议

### Anki 设置优化
1. **新卡片**：每天学习20-30个新单词
2. **复习间隔**：使用Anki默认的间隔重复算法
3. **学习时间**：建议每天固定时间学习，保持连续性

### 学习方法
1. **看到英文想中文**：测试理解能力
2. **听发音练口语**：可以在Anki中添加音频
3. **造句练习**：在笔记中添加例句
4. **定期复习**：利用Anki的复习提醒功能

## 常见问题

**Q: 导入后中文显示乱码怎么办？**
A: 确保在导入时选择UTF-8编码，如果仍有问题，可以用记事本打开CSV文件另存为UTF-8格式。

**Q: 可以重复导入吗？**
A: 可以，但Anki会检测重复内容。建议每次导出新增的单词，或者在导入前清理重复项。

**Q: 如何在Anki中添加真人发音？**
A: 可以使用Anki的AwesomeTTS插件，或者手动添加音频文件。

**Q: 导出的文件在哪里？**
A: 通常在浏览器的默认下载文件夹中，文件名格式为 vocabulary_YYYY-MM-DD.csv。

## 技术说明

- 导出格式：UTF-8编码的CSV文件
- 字段分隔符：逗号
- 文本限定符：双引号
- HTML支持：背面字段支持基本HTML标签（如`<br>`、`<i>`等）
