# 英文悬浮翻译 Chrome 插件

一个简单易用的Chrome浏览器插件，可以在鼠标悬浮到英文单词时自动显示中文翻译和读音。

## 功能特点

- 🖱️ **鼠标悬浮翻译**：将鼠标悬浮在英文单词上即可显示翻译
- 🔊 **语音朗读**：点击发音按钮或开启自动发音功能
- 💾 **智能缓存**：已翻译的单词会被缓存，提高响应速度
- ⚙️ **个性化设置**：可以开启/关闭翻译功能和自动发音
- 🎨 **美观界面**：简洁的悬浮窗设计，不干扰正常浏览

## 安装方法

### 开发者模式安装

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 插件安装完成！

## 使用方法

1. **基本翻译**：
   - 在任意网页上将鼠标悬浮在英文单词上
   - 等待1-2秒，翻译窗口会自动显示
   - 移开鼠标，翻译窗口自动隐藏

2. **语音朗读**：
   - 在翻译窗口中点击🔊按钮听取单词发音
   - 或在设置中开启"自动发音"功能

3. **插件设置**：
   - 点击浏览器工具栏中的插件图标
   - 可以开启/关闭翻译功能
   - 可以开启/关闭自动发音功能

## 技术实现

- **翻译API**：使用Google翻译的免费API
- **语音合成**：使用浏览器内置的Web Speech API
- **缓存机制**：使用Map对象缓存翻译结果
- **响应式设计**：适配不同屏幕尺寸

## 注意事项

- 插件需要网络连接才能获取翻译结果
- 某些网站可能因为CSP策略限制翻译功能
- 建议在设置中根据个人需要调整自动发音功能
